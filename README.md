# fano-7b-homoeostat
«Asymmetric 7-bit cellular automaton for topological phase stabilization, jitter utilization and invariant retention over GF(2) and complex fields.»(Асимметричный 7-битный клеточный автомат для топологической стабилизации фазы, утилизации джиттера и удержания инварианта над полем GF(2) и комплексными полями).
